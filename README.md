## Code Details

- Frontend: React.js
- Styling: Bootstrap 4
- Visualizations: p5.js
- Client-side routing: react-router
- Test runner: Mocha
- Assertion library: Chai
- Bundler: Webpack
- Transpiler: Babel
- Package manager: npm
- Backend: Node.js & Express (Firebase Functions)
- Storage: Firebase Firestore
- Hosting: Firebase Hosting
